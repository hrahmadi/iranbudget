# Iran Budget Visualization

Interactive Sankey diagram visualization of Iran's national budget data (1395-1404 / 2016-2025).

## Features
- 10 years of budget data
- Interactive Sankey diagrams
- Multi-language support (English/Farsi)
- Currency conversion (Rial/Dollar)
- Year selection

## Tech Stack
- React + TypeScript
- PostgreSQL database
- Python scripts for data processing

## Setup
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Data
All monetary values in billion rials. Source data in `/data` directory.
